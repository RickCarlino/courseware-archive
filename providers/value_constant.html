Constant vs. Value

  • Constant can be injected at .config() time
  • Constant CANNOT be overridden by a $decorator
  • Values and constants CAN be changed at run time if they are an OBJECT.
  • Values and constants CAN'T be changed at run time if they are a PRIMITIVE.
