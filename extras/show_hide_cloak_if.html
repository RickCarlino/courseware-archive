When are ng-show/ng-hide faster than ng-if?

ng-show and ng-hide might be faster than ng-if in cases where initializing elements is more expensive than rendering them.

Example: putting an ng-controller in an ng-if will re-initialize the controller every time.

Like all performance optimizations, you will need to profile performance to prove that it actually improves performance for your use case.
